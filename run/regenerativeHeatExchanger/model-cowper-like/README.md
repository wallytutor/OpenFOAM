# Cowper-like exchanger


This model represents a Cowper-like regenerative heat exchanger, where the hot and cold fluids alternate in a single channel during charging and discharging cycles, respectively. It is actually represented by 1/6th of the actual honeycomb structure.

About the conception:

- As the geometry is generated with `gmsh`, one needs to take care to keep the cyclic boundaries conformal, otherwise `cyclicAMI` conditions may be required and are currently not included in the present model.

- The multi-region geometry generated by `gmsh` needs to have all bodies and surfaces named as physical groups; the only exception being the coupled surfaces, which are not named and identified by their topology.

- It must be emphazised that surfaces and volumes cannot share a tag number, *i.e.* if a volume is indexed as `1`, there should be no surface `1` in the model, otherwise the conversion with `gmshToFoam` will fail to keep names.

- Mesh must dumped to file in format MSH 2.2.

- Original model was adapted from [multiRegion/CHT/heatedDuct](https://github.com/OpenFOAM/OpenFOAM-13/tree/master/tutorials/multiRegion/CHT/heatedDuct) as it was probably the closest match to our goals here.

- Additional elements were eventually based on other [CHT tutorials](https://github.com/OpenFOAM/OpenFOAM-13/tree/master/tutorials/multiRegion/CHT), what includes material properties and base setup of turbulence.

- After conversion, the mesh is renumbered with `renumberMesh` for fixing face orientation (to be verified). Finally, mesh is split into regions with `splitMeshRegions` utility, and the original `constant/polyMesh` directory is removed to avoid confusion.

## Generate geometry

```python
mesh = "mesh.msh"

yaml = YAML()
data = yaml.load(open("dimensioning.yaml"))

geom = CowperLikeGeometry(
    m_h = data["m_h"],
    D_h = data["D_h"],
    h_t = data["h_t"],

    # Use with care based on the above values:
    num_points_angular = 6,
    fluid_bl_tot = 0.003,
    fluid_bl_ext = 0.0001,
    fluid_bl_int = 0.0006,
    solid_bl_ext = 0.0050,
    solid_bl_int = 0.0001,
    rel_layer    = 0.15,
)

if not Path(mesh).exists():
    geom.create_model(saveas=mesh, render=True)
```

## Prepare mesh

```python
def run(log, cmd):
    if Path(log).exists():
        print(f"Already run, check log {log} for details")
        return

    with open(log, "w") as f:
        print(f"Logging to {log}: {' '.join(cmd)}")
        subprocess.run(cmd, stdout=f, stderr=f, check=True)
```

```python
log1 = "log.1.gmshToFoam"
log2 = "log.2.renumberMesh"
log3 = "log.3.createPatch"
log4 = "log.4.splitMeshRegions"
log5 = "log.5.checkMeshFluid"
log6 = "log.6.checkMeshSolid"

run(log1, ["gmshToFoam", mesh])
run(log2, ["renumberMesh"])
run(log3, ["createPatch", "-overwrite"])
run(log4, ["splitMeshRegions", "-cellZones"])
run(log5, ["checkMesh", "-region", "fluid"])
run(log6, ["checkMesh", "-region", "solid"])

# To avoid any confusion, remove original mesh:
if Path("constant/polyMesh").exists():
    shutil.rmtree("constant/polyMesh")
```
